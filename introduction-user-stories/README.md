## User Stories

User stories are a great way to break down what the different features of the website are, and how to structure it. They are generally written out in this form "As a user, I want to...". 

> 🧠 Remember formatting your user stories as smaller, more focused stories can make the development process easier to manage.

Here are some user stories for our lab:

- As a user, I want to be able to select numbers so that they can do things with them.
- As a user, I want to be able to add numbers together.
- As a user, I want to be able to subtract numbers.
- As a user, I want to be able to divide numbers.
- As a user, I want to be able to multiply numbers together.
- As a user, I want to be able to see the output of the mathmatical operation.
- As a user, I want to be able to clear all operations and start from 0.
